> **현재 주가**: $31.05 (2026-05-29 종가, 당일 +11.2% 급등)
> **목표 주가**: $28 ~ $35 (애널리스트 평균 목표가 $30.3, 중앙값 $28)

### 1.2 비즈니스 모델
GitLab은 **소프트웨어 개발 생명주기 전체(DevSecOps)를 하나의 플랫폼**으로 제공하는 SaaS 기업이다.
기획 → 코드 작성 → 보안 스캔 → 컴플라이언스 → 배포까지 **단일 애플리케이션**에서 통합 관리한다.

- **구독(Subscription) 기반 매출**: SaaS(클라우드) + Self-Managed(고객 자체 설치) 형태
- 매출의 **약 70%가 Self-Managed** → AI 신기능 채택에 다소 시차가 발생하는 구조
- 핵심 경쟁자: **Microsoft GitHub** (최대 경쟁자), Atlassian, JFrog 등

### 1.3 AI 전략 — GitLab Duo
**GitLab Duo Agent Platform**: 대화형 AI 어시스턴트 + 특화 AI 에이전트가 기획·개발·보안·배포 작업을
자율적으로 수행하도록 오케스트레이션. AI 코딩 시대에 GitHub Copilot 등과 직접 경쟁하는 핵심 성장 동력.
다만 FY27 가이던스는 **Duo 매출 기여를 최소로 가정** → 아직 본격 수익화 전 단계.

## 2. 재무 분석

### 2.1 FY2026 (2026년 1월 종료) 실적 — 고성장 + 흑자 기반 확보
| 항목 | 값 | 비고 |
|------|------|------|
| 매출 | **$955.2M** | 전년 $759.3M 대비 **+25.8%** |
| 연간반복매출(ARR) | **$10억 돌파** | SaaS 기업의 핵심 지표 |
| 매출총이익률 | 약 **87%** | 전형적 고마진 SaaS |
| Non-GAAP 영업이익률 | **+17%** | 수익성 본궤도 진입 |
| GAAP 영업이익률 | **−7%** | 주식보상비용(SBC) 영향 |
| Non-GAAP 잉여현금흐름(FCF) | **$219.6M** (마진 ~23%) | 견조한 현금 창출력 |
| GAAP 순손익 | 약 **−$56M** (EPS −$0.34) | SBC로 GAAP은 적자 |

### 2.2 핵심 SaaS 지표 (건강성 점검)
| 지표 | 값 | 해석 |
|------|------|------|
| $10만+ 고객 수 | **1,456개** (전년比 +18%) | 대형 고객 확대, ARR의 75%+ 차지 |
| 달러기반 순매출유지율(DBNRR) | **약 121%** (Q2 FY26) | 기존 고객이 매년 21%씩 더 지출 (우수) |
| 매출총이익률 | 87% | SaaS 최상위권 |

> DBNRR 121%는 "기존 고객만으로도 매년 21% 성장"을 의미 → SaaS 비즈니스 건강성 양호.
> 다만 과거 130%대에서 둔화된 추세는 주의 필요.

### 2.3 재무 건전성
- **무차입 + 현금 $12.6억** → 재무구조 매우 견고, 자사주 매입($4억 규모 승인) 여력 충분
- GAAP 적자의 주원인은 **주식보상비용(SBC)** — 현금 유출이 아닌 회계상 비용. 실질 현금흐름은 흑자.

## 3. 밸류에이션

### 3.1 주요 지표 (실시간 기준)
| 지표 | 값 | 비고 |
|------|------|------|
| 주가 | $31.05 | 52주 밴드 $18.73 ~ $52.38 |
| 시가총액 | 약 $5.24B | |
| P/S (TTM) | 약 **5.5배** | 과거 대비 크게 낮아진 수준 |
| 선행 PER | 약 **32배** | Non-GAAP 이익 기준 |
| EV/Revenue | 약 4.3배 | 순현금 반영 시 |
| P/B | 약 5.3배 | |
| 잉여현금흐름 수익률 | 약 **4.2%** | FCF $219.6M / 시총 |
| 배당 | 없음 | 성장 재투자 |

### 3.2 밸류에이션 해석
GitLab은 **고성장 SaaS** → 매출 멀티플(P/S)과 잉여현금흐름이 핵심이다.
- 현재 **P/S 약 5.5배**는 IPO 직후 20배 이상이었던 것에 비하면 **크게 디레이팅(de-rating)** 된 수준.
- 주가는 52주 고점 $52에서 **−41%** 하락, 고점 대비 크게 조정된 상태.
- 성장률 26%·FCF 마진 23%를 감안하면 **"비싸지 않은 고성장주"** 영역에 진입.
- 다만 **숏 비중이 유통주식의 약 13%**로 높아, 시장의 의구심(성장 둔화·AI 경쟁)이 상당함을 시사.
- 당일 +11% 급등은 단기 변동성 — **6/3 실적 발표 결과에 따라 방향성 결정**.

## 4. 경쟁 환경

| 항목 | GitLab (GTLB) | GitHub (Microsoft) | Atlassian |
|------|--------|---------|---------|
| 포지션 | 통합 DevSecOps 단일 플랫폼 | 개발자 협업·코드 호스팅 1위 | 협업·이슈관리(Jira) |

| 강점 | 단일 플랫폼 통합성, 보안·컴플라이언스 | 생태계·MS 번들링·Copilot 선점 | 워크플로우 점유율 |

> GitLab은 "단일 통합 플랫폼"과 "보안·컴플라이언스 강점"으로 차별화 중.

## 5. 투자 포인트

### 5.1 강세 요인 (Bull Case)
1. **고성장 + 수익성 동시 달성** — 매출 +26%, Non-GAAP 영업이익률 17%, FCF 마진 23%
2. **ARR $10억 돌파 + 대형고객(+18%)** — 엔터프라이즈 침투 가속, DBNRR 121%
3. **밸류에이션 디레이팅** — P/S 5.5배로 고점 대비 크게 낮아져 진입 매력 개선
4. **무차입·현금 $12.6억** — 자사주 매입($4억) 여력, 매크로 충격 방어력
5. **AI(Duo) 옵션 가치** — 가이던스에 거의 미반영 → 성공 시 업사이드 서프라이즈 여지

### 5.2 약세 요인 (Bear Case)
1. **성장률 둔화** — FY27 가이던스 +15~17%로 감속 (FY26 +26% → 둔화)
2. **GitHub/Copilot 경쟁 심화** — MS의 자본력·번들·AI 선점이 구조적 위협
3. **AI가 양날의 검** — AI 코딩이 "개발자 좌석 수" 기반 과금 모델을 잠식할 위험
4. **GAAP 적자 + 높은 SBC** — 주식보상에 따른 희석 지속
5. **높은 숏 비중(유통 13%)** — 시장의 강한 의구심 반영

## 6. 리스크 요인

1. **경쟁 리스크**: Microsoft GitHub+Copilot의 압도적 자본·번들링 전략
2. **AI 파괴적 전환 리스크**: 생성형 AI가 좌석 기반 SaaS 과금 모델 자체를 흔들 가능성
3. **성장 둔화 리스크**: 가격 민감 고객군(ARR의 ~20%) 압박, 성장률 감속
4. **밸류에이션 리스크**: 여전히 선행 PER 32배 → 성장 미스 시 추가 디레이팅
5. **희석 리스크**: 높은 주식보상비용(SBC)에 따른 지속적 주식 수 증가
6. **이벤트 리스크**: 6/3 실적 발표 — 가이던스/AI 진척에 따라 급등락 가능

## 7. 투자 결론

### 7.1 종합 의견
GitLab은 **"고성장에서 고성장+수익성으로 전환 중인 우량 SaaS"** 다. 매출 +26%, ARR $10억 돌파,
Non-GAAP 영업이익률 17%, FCF 마진 23%, 무차입에 현금 $12.6억 — **펀더멘털은 견조**하다.
밸류에이션도 IPO 직후 거품(P/S 20배+)에서 **P/S 5.5배로 크게 정상화**되어 진입 매력이 개선됐다.

그러나 ① 성장률이 FY27에 **15~17%로 둔화**되고, ② **Microsoft(GitHub+Copilot)** 와의 경쟁이
구조적 위협이며, ③ AI가 좌석 기반 과금 모델을 잠식할 수 있다는 **양날의 검** 성격을 갖는다.
유통주식의 약 13%에 달하는 높은 숏 비중이 이런 시장의 의구심을 대변한다.

결론적으로 GitLab은 **장기 투자 매력이 있는 고성장주**이나, 단기적으로는 **6월 3일 실적 발표라는
큰 이벤트가 임박**해 있다. 따라서 신규 진입은 **실적·가이던스(특히 Duo AI 수익화·DBNRR 추세) 확인 후**가
위험 대비 합리적이다. 애널리스트 컨센서스도 **'보유'**(강력매수1/매수7/보유17/매도0/강력매도2) 우위다.

### 7.2 목표주가 (시나리오)
- Bear (P/S 4배, 성장 둔화·경쟁 심화): **약 $24**
- Base (P/S 5.5배, 현 수준 유지): **약 $30** (애널리스트 평균 $30.3와 부합)
- Bull (P/S 7배, AI Duo 수익화 가시화): **약 $40~52** (애널리스트 최고 $60)
- → **목표주가 밴드 $28 ~ $35** (현 주가 $31은 Base 영역)

### 7.3 투자 체크리스트
- [ ] **6/3 FY27 Q1 실적** — 매출/가이던스가 컨센서스($253~255M) 부합·상회 여부
- [ ] **DBNRR 추세** — 121%에서 추가 둔화 여부 (둔화 지속이면 경계)
- [ ] **GitLab Duo(AI) 수익화** 진척 — 가이던스 상향 트리거 가능성
- [ ] **Self-Managed → SaaS/AI 업그레이드 램프** 진행 상황
- [ ] GitHub Copilot과의 경쟁 구도 변화
- [ ] 숏 비중·SBC 추이 점검
